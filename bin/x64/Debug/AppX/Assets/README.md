# UWP Asset Placeholders

This directory contains placeholder files for UWP packaging assets.
Replace these with actual images before submitting to the Microsoft Store or deploying to Xbox One.

Required assets (referenced in Package.appxmanifest):
- `StoreLogo.png` — 50x50 Store listing logo
- `Square150x150Logo.png` — 150x150 tile logo
- `Square44x44Logo.png` — 44x44 app list logo
- `Wide310x150Logo.png` — 310x150 wide tile
- `SplashScreen.png` — 620x300 splash screen
- `LockScreenLogo.png` — 24x24 lock screen badge

For Xbox One, also consider:
- `Square480x480Logo.png` — Xbox dashboard tile
- `Wide960x480Logo.png` — Xbox wide tile
